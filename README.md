# 🕹️ GAMES

> **My playground for exploring game development concepts by building everything from scratch.**

This repository is a curated collection of games I have developed and continue to develop; to explore game mechanics, logic design, and different programming paradigms.

The main goal is to understand how core systems work internally by implementing them from the ground up rather than relying heavily on external frameworks or engines.

This project also serves as a practical space to experiment with:

* Procedural and functional programming methodologies.
* Object-oriented programming (OOP).
* Transitioning codebases from procedural design to OOP architecture.
* Game state management and rendering logic.
